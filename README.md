### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/nguyenhieu1435/LingolandTOEIC.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run app
   ```sh
   npm start > (press a to open Android | press w to open Web)
   ```
